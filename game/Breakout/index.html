About Breakout

Breakout is a classic arcade game where you control a paddle and bounce a ball to destroy all the blocks. Keep the ball in play, clear the board, and try to get the highest score.

How to Play

Use the Left Arrow and Right Arrow keys to move the paddle. Press Space or Start Game to begin. On mobile, use the on-screen buttons to move left, move right, and start the game.
